1) Домашнее задание
Реализация пейджинга на реальном API

Описание/Пошаговая инструкция выполнения домашнего задания:

- Используйте открытое API: https://github.com/public-apis/public-apis

- Добавить несколько рубрик по разным запросам (например, новости или города для отображения погоды). Переключение между рубриками реализовать через горизонтальный ScrollView или SegmentedControl.

- При переключении рубрик содержимое List должно обновляться, а пейджинг работать корректно.

- Реализовать навигацию в глубину до 3 экранов с использованием кастомного навигационного стека.

- При выборе ячейки на каждом экране (разделе SegmentedControl) добавить анимацию, имитирующую улет ячейки по кривой вниз (например, как анимация сохранения в избранное).

2) Домашнее задание
Создание модуляризированного приложения

Описание/Пошаговая инструкция выполнения домашнего задания:
- Выполнить рефакторинг приложения. При реализации на SwiftUI использовать стандартные механизмы: Combine, Signals или EventBus.
- Модульно организовать приложение, вынеся UI-компоненты в отдельный модуль и импортируя их в местах использования.
  
Создать:
- ServiceLocator (для SOA).
- Вложенные ViewModel (для MVVM).
- Перевести существующие сервисы на модульную архитектуру.
- Добавить инжекцию зависимостей в переменные классов для повышения их читабельности.
- Выделить уровень Core-сервисов (сеть, парсинг, хранение данных).
- Вынести Core и другие сервисы в отдельный SPM модуль.
